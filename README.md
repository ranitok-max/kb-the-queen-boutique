# KB The Queen Boutique — Loja Completa

Loja estática pronta para publicar no Vercel.

## O que inclui

- 5 produtos
- Galerias com frente, costas, detalhes e tecido
- Vídeos dos produtos
- Seleção de cor e tamanho
- Carrinho persistente
- Finalização da encomenda pelo WhatsApp
- Layout responsivo para telemóvel e computador

## Publicar no GitHub/Vercel

1. Descompacta este ficheiro.
2. No teu repositório `kb-the-queen-boutique`, apaga os ficheiros antigos.
3. Faz upload de **todos os ficheiros e da pasta `assets`**.
4. Faz `Commit changes`.
5. O Vercel atualiza automaticamente em poucos minutos.

## Alterar produtos

Edita o ficheiro `products.json`.
