# KB The Queen Boutique

Loja estática pronta para publicar no Vercel.

## Como publicar

1. Descompacta a pasta.
2. Entra em vercel.com.
3. Cria um novo projeto.
4. Faz upload desta pasta ou liga ao GitHub.
5. Publica.

## Ficheiros principais

- index.html: página inicial e catálogo
- produto.html: página individual do produto
- carrinho.html: carrinho com finalização por WhatsApp
- styles.css: cores e design
- script.js: produtos, carrinho e WhatsApp

## Personalizações importantes

No ficheiro `script.js`, altera:
- nomes dos produtos
- preços
- fotografias
- descrições

Procura o número `447404824408` e substitui pelo WhatsApp oficial da boutique, se necessário.

Esta versão não processa pagamentos diretamente. As encomendas são finalizadas por WhatsApp.
